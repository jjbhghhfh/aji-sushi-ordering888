## 🚀 **快速部署（3种方法）**

### **方法1：Vercel（最推荐）**
1. 访问 https://vercel.com/new
2. 导入这个项目（通过GitHub或上传）
3. 点击 Deploy
4. **完成！获得网址！**

### **方法2：Netlify**
1. 访问 https://app.netlify.com/
2. 导入项目或拖拽 `dist` 文件夹
3. **完成！获得网址！**

### **方法3：使用 CLI（最快）**
```bash
npm install -g vercel
cd vercel-deploy
npm install
vercel --prod
```

📖 **详细步骤请查看：[DEPLOY.md](./DEPLOY.md)**
